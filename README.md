## 下载

1. 在仓库根目录打开 **`S5-Residency-Countdown.html`**  
2. 点页面右上角 **Raw**  
3. 用浏览器直接打开，或「另存为」到本地再打开即可。

---

## 技术说明

- **单文件**：HTML 内已含样式与脚本，无需构建。  
- **字体**：使用 Google Fonts，首次打开需能访问外网。  
- **时间与逻辑**：均写在该文件中；改期、改起止，只动这一处里配置区与相关注释所指的代码。
